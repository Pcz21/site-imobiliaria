## 🚀 Sobre o Projeto

O Fabiju Imóveis é uma plataforma imobiliária desenvolvida para gerenciamento e divulgação de imóveis.

O sistema possui:

- Cadastro de imóveis
- Upload de imagens e vídeos
- Painel administrativo
- Autenticação JWT
- Integração com SQL Server
- Integração com Supabase Storage
- Responsividade Mobile
- Sistema de visualizações
